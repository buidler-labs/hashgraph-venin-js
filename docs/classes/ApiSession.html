    • Defined in lib/ApiSession.ts:89

    Given a raw JSON {@link object}, it triest ot upload it using the currently configured {@link Client} passing in-it any provided {@link args}.

    Note: This is the same as calling the more verbose equivalent of upload(new Json(what)).

    Example of usage:

    await apiSession.upload({a: 1, {b: "c"}})

    Parameters

    • what: object

      The Json-acceptable payload to push through this ApiSession

    • Rest ...args: any[]

      A list of arguments to pass through the upload operation itself. Note: this list has, by convention, at various unpaking stages in the call hierarchy, the capabilities to specify SDK behaviour through eg. "_file" ({@link Uploadable}) or "_contract" (Contract)

    Returns Promise<LiveJson>

    • An instance of the associated LiveJson resulting {@link LiveEntity}.
